# Banking System - Java

## Project Description

This project is a simple banking system developed in Java to simulate basic banking operations.

The purpose of this project is to practice Object-Oriented Programming concepts and software development logic.

## Features

- Account creation
- Deposit
- Withdraw
- Balance checking

## Technologies

- Java
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository
2. Open the project in NetBeans or VS Code
3. Run the Main.java file
